# Budget Buddy Website

This repository contains a modern static landing page for Budget Buddy, ready to deploy on GitHub Pages.

## Files

- `index.html`: main page content
- `styles.css`: complete visual design and responsive layout
- `script.js`: small UI behavior (footer year + reveal effect)
- `assets/images/*.svg`: placeholder images you can replace anytime

## Replace placeholders

Swap these files with your own screenshots/images (keep the same names or update paths in `index.html`):

- `assets/images/hero-dashboard.svg`
- `assets/images/phone-mockup.svg`
- `assets/images/app-preview.svg`

You should also update:

- section text and testimonials in `index.html`
- App Store / Google Play links in `index.html` (`#download` section)

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Under **Build and deployment**, select:
   - `Source`: **Deploy from a branch**
   - `Branch`: **main** (root)
4. Save and wait for deployment.
5. Your site will be available at your GitHub Pages URL.
